Mit dem Plugin Asylgesetz-Referendum wird ein in der linken Ecke fixierter Link zur Homepage des Asylgesetz-Referendums 2012 eingebunden.

== Description ==

Es wird ein in der linken Ecke fixierter Link zur Homepage des Referendums http://www.asyl.ch eingebunden. Code basiert auf dem Plugin wp-bullerjahn-2011, wurde angepasst.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

1. Pluginordner entzippen und den Inhalt in das Verzeichnis `/wp-content/plugins/` hochladen.
2. Plugin via Adminpanel aktivieren.
3. Fertig!


== Frequently Asked Questions ==

= Kann ich das Bild auswechseln oder den Link ändern? =

Nein, diese sind in dieser Version fest programmiert.
